Anime Scraper API Documentation

API Endpoints

There are two main endpoints:

  • /api/stream: Returns the streaming information as a JSON object.
  • /api/embed: Returns an embeddable player.

Query Parameters

  • anilistId (required): The AniList ID of the anime.
  • episode (required): The episode number.
  • server (required): The streaming server. Available servers can be found by inspecting the network requests on hianime.do. Common servers include 'HD-1', 'HD-2'.
  • type (required): The stream type, either sub or dub.
